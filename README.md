# Futtips_LP1

Trabalho de LP1 - FATEC SJRP - Lojas FutTips

## Descrição

Este projeto é um sistema para gerenciamento de pedidos e vendas de camisas da loja FutTips. Ele foi desenvolvido como parte do trabalho da disciplina de Linguagem de Programação 1 (LP1) da FATEC São José do Rio Preto.

## Estrutura do Projeto
src/
app/
Main.java
entities/
Camisas.java
Cargos.java
Clientes.java
Enderecos.java
Funcionarios.java
Parcelas.java
Pedidos.java
PedidosCamisas.java
Pessoas.java
Tipos.java


- **app/Main.java**: Ponto de entrada da aplicação. (Não está sendo utilizado)
- **entities/**: Contém as classes de entidades do sistema, como Camisas, Pedidos, Funcionários, etc.

## Como Executar

O sistema não está pronto para execução.

## Funcionalidades
Cadastro e gerenciamento de camisas, pedidos, clientes e funcionários.
Associação entre pedidos e camisas.
Estrutura orientada a objetos, facilitando manutenção e expansão.
